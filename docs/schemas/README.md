- `2_technical_architecture_modules_and_channels.md`

Latest updates include:
- native `.vncloc` direct TCP console path
- admin usage analytics coverage in portal structure diagrams
- gold image management in portal structure (admin backend)

## PPT Usage

You can:

1. Open each file in Cursor/Markdown viewer and export snapshot images.
2. Copy the Mermaid blocks into any Mermaid-capable renderer and export PNG/SVG.
3. Insert exported images into PowerPoint.
